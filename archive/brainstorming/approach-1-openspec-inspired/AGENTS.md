# approach-1-openspec-inspired/ orientation

**Abandoned design direction.** Reference-only.

This directory holds an early OpenSpec-inspired proposal that was set
aside in favor of the NLSpec-based approach in
`../approach-2-nlspec-based/`. The bootstrap plan does NOT consume
anything from this directory.

## Files

| File | Status |
|---|---|

## Why archived rather than deleted

shaped `approach-2-nlspec-based/`. Keeping the artifacts means future
readers can understand why the project landed where it did.
